# Raccourcis clavier

Cette page récapitule tous les raccourcis disponibles dans **Automask**.

---

## Navigation
- **← / ↑** : Image précédente
- **→ / ↓** : Image suivante
- **Molette souris** : Zoom avant / arrière
- **Clic droit + glisser** : Déplacement (pan)

---

## Annotation
- **Clic gauche** : Ajouter un point positif
- **Ctrl + Z** : Annuler le dernier point
- **Ctrl + X** : Supprimer tous les points

---

## Masque
- **Ctrl + M** : Calculer le masque
- **Ctrl + I** : Inverser le masque calculé
- **Ctrl + D** : Supprimer le masque
- **Espace** : Afficher / masquer le masque
